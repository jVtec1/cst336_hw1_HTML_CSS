Programming Languages

A computer programming language is any set of detailed instructions for a digital computer. These instructions can then be directly executed by the computer when in a form known as machine language, 0s and 1s. Programming language written in Assembly language or a higher level language can be translated down in this machine language. There are many different types of programming languages, Object Oriented, Machine and Assembly languages, Algorithmic, Buisiness-Oriented, Education-Oriented, Declarative, Scripting, Document formatting, and World Wide Web display languages. There are many computer programming languages that exist today, but only a few are widely used. Three of the most common and heavily taught languages are all Object Oriented languages, these are Java, C++, and Python.
